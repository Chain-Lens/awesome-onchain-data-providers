# Launch Blurb

## Short

We just opened Awesome Onchain Data Providers: a GitHub-native, machine-readable directory for RPCs, indexers, oracles, analytics APIs, subgraphs, and specialized onchain data services.

Add your provider with a PR. If you want agents to call and pay for your API, use the ChainLens registration flow after the directory entry is merged.

## Social Post

Launching a GitHub-native directory for onchain data providers:

https://github.com/pelican-lab/awesome-onchain-data-providers

RPCs, indexers, oracles, analytics APIs, subgraphs, and specialized data services, all as structured JSON.

Provider teams can submit a PR for discovery. Teams that want executable paid API listings can continue through ChainLens registration.

## Provider Outreach

We're building a machine-readable directory of onchain data providers and added your project as a seed entry.

If anything is outdated, a PR is welcome. If you want your API to be executable by agents through ChainLens, the directory now supports an optional `chainlens` intent block and links to the ChainLens registration flow.
